Mini Arduino CNC

El arduino CNC fue hecho con un Arduino UNO y puede usarlo para fresar madera delgada, diferentes espumas o puede usarlo para trazar diferentes bocetos y también se puede usar para fines de grabado y grabado.

Materiales Necesarios

  • 3 CD/DVD-Roms antiguos (para los motores paso a paso)
  • 1 Fuente de alimentación de computadora
  • 3 Tablas Easydriver
  • 1 Arduino UNO
  • 2 piezas de madera
  • 1 Soldador
  • 1 Alambre de soldar

Pasos Básicos

  • Mira el video de demostración para ver el CNC en acción.
  • Desmonta 3 unidades de CD/DVD-Rom para extraer los motores paso a paso necesarios para los ejes X, Y y Z.
  • Arma una base de madera y fija los pilares de aluminio. Ensambla los ejes X y Y asegurándote de que estén perpendiculares entre sí.
  • Suelda los cables a los motores paso a paso y conéctalos a las tablas Easydriver. Usa un soldador de baja potencia para evitar daños.
  • Descarga e instala el software Arduino y la biblioteca Grbl. Sube la biblioteca a tu Arduino y configura el controlador Grbl para ajustar las opciones y direcciones de los motores. Usa Inkscape para generar Gcodes y prueba con DeskEngrave para crear tus diseños.
